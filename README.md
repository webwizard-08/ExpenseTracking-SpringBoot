 # Expense Tracker - Full Stack Project

---

# 🚀 Project Overview

Expense Tracker is a full-stack web application developed using:

- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

This project helps users to:

- Add Expenses
- View Expenses
- Update Expenses
- Delete Expenses

The application follows proper backend architecture using:

- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer

---
